## zh

# 核心原则

## 删前保后

后说的通常更完整，所以：
- 删除前面的（口误/残句）
- 保留后面的（完整句）

## 例外

用户明确说"后面更顺"时，删前保后。
用户明确说"前面更顺"时，删后保前。

## en

# Core Principles

## Delete Earlier, Keep Later

The later version is usually more complete, so:
- Delete the earlier part (slips / fragments)
- Keep the later part (complete sentence)

## Exceptions

When the user explicitly says "the later version flows better" → delete earlier, keep later.
When the user explicitly says "the earlier version flows better" → delete later, keep earlier.
